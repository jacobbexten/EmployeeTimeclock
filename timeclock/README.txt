Timeclock App within the project that contains the clock in/out feature, entries, and calendar. It contains models.py that has the author, time, date, notes, and options of clocking in or out. It's urls.py contains all the necessary urls associated with the correct html files and views associated with them. The views.py contatins all the necessary views connected with their respective models. The forms.py extends the models.py for the options of clocking in and out. The tests.py contains tests.
